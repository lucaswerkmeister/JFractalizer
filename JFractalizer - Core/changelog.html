Full changelog for the JFractalizer:
  • v1.1 (26.09.2012):
    • New core feature: Change FractalProvider
    • New fractal: Julia Set
    • New Mandelbrot feature: Switch to according Julia set
  • v1.0 (16.09.2012):
    • Removed the split of the core into two packages
    • New core feature: "Core" class for access to all core funcionality from plugins
    • Fixed a core bug where the ClassChooserDialog would crash if no implementation was found
    • New core feature: FractalProviders have to provide a zoom method
    • New core feature: Context menu for canvas with zoom options
    • New core feature: FractalProviders can add items to the canvas' context menu
    • New Mandelbrot feature: "Go to start image" context menu option
    • New core feature: "Exit" menu item
    • New policy: Fractals shall use keyboard shortcuts without the Shift modifier, Color Palettes shall use keyboard shortcuts with the Shift modifier.
    • New Mandelbrot feature: Undo/Redo.
    • Fixed a Mandelbrot bug that would not show the selection area.
    • Fixed several core bugs in file opening.
    • New core feature: Status bar
    • New Mandelbrot feature: Shows calculation time in status bar
  • v0.4 (28.07.2012):
    • Changed the internal Fast Storage Lists of SimplePalette and NodePalette to ArrayLists, should improve performance
    • New core feature: FractalProviders and ColorPalettes can now customize their menus
    • New Mandelbrot feature: Added "Edit Boundaries..." option
    • Fixed a core bug where closing the ClassChooserDialog with the OS "X" button would crash the JFractalizer
    • New Mandelbrot feature: Added "Recalculate" option
    • New SimplePalette feature: "Edit Palette" Menu option and dialog
    • New Mandelbrot feature: Added "Edit additional parameters..." option
    • New NodePalette feature: "Edit Palette" Menu option and dialog
    • Fixed a Mandelbrot bug where zooming in could make the depth decrease, reworked the algorithm that adapts the depth
    • New core feature: Global JColorChooser
  • v0.3 (06.07.2012):
    • Added a "Choose implementation" dialog to choose between implementations of interfaces
    • Reorganized the code into the core (plugin loader, GUI) and the default plugin (Mandelbrot, SimplePalette, NodePalette)
    • Program will now at startup ask for Fractal and ColorPalette to use
  • v0.2 (01.07.2012):
    • Changed the window title to "JFractalizer"
    • Added license text to all files; added license to project
    • Mandelbrot now utilizes a×b processor cores instead of n×n cores.
    • Moved all packages into the superpackage de.lucaswerkmeister.code.jfractalizer
    • New Feature: Can navigate the Mandelbrot Set with the mouse. Drag to choose new area. Selected area is drawn inverted.
    • Bugfix: ColorPalettes would not be loaded properly
    • Several performance enhancements in the MandelbrotImageMaker:
      • Removed unnecessary synchronization around Palette.getColor()
      • Moved commands out of synchronized(Graphics) block
      • Moved calculations with constant results out of the pixel loop
  • v0.1 (30.06.2012):
    • First published version.
    • Can render arbitrary regions of the Mandelbrot Set using a custom color palette.
    • Supports anti-aliasing and multi-core calculation.
    • Can save the rendered image to a png-, gif- or jpg-image.
    • Can save and load fractal setups.
